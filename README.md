# 🎬 CineBusca

Aplicação **React + Vite** que consome a **OMDb API** para buscar filmes, ver **detalhes** (diretor, elenco, sinopse, avaliação) e montar uma **lista de favoritos** persistida no `localStorage`.

## 🚀 Rodando localmente

```bash
npm install
npm run dev
